## 🧱 Phase 1：岗位信息采集（Job Data Collection）

> 🎯 目标：**把招聘网站变成“我自己的岗位数据库”**

### 1.1 数据来源定义

* [ ] 明确首选招聘平台（如 Boss 直聘）
* [ ] 明确岗位范围

  * [ ] 技术方向（如 Java / 后端 / 架构）
  * [ ] 城市（如 北京）
  * [ ] 年限范围（如 5–10 年）

---

### 1.2 RPA / 自动化抓取

* [ ] 选定抓取方式

  * [ ] RPA 工具（影刀 / UiBot）
  * [ ] 或 Playwright / Selenium
* [ ] 实现最小抓取流程

  * [ ] 搜索岗位
  * [ ] 打开岗位详情
  * [ ] 复制 JD 原文
* [ ] 增加基础风控控制

  * [ ] 手动触发
  * [ ] 限制抓取数量（≤20 / 次）
  * [ ] 增加随机等待

---

### 1.3 原始数据存储（Raw Layer）

* [ ] 设计原始表 `job_raw`

  * [ ] source（来源）

  * [ ] jd_text（原始 JD）
  * [ ] crawl_time
* [ ] 确保 **原始文本永不覆盖**
